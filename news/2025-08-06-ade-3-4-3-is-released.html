ADE 3.4.4 Is Released!

Aug 6, 2025

Hsinchu, Taiwan – Immense Oak Technologies, Inc. (IOT) released ADE version 3.4.4.

This ADE release focuses on the project download speed and power on speed. When users start to use ADE to design HMI with extensive set of widgets, download speed is slow. With this ADE release, users can expertise a faster download and power-on. The ADE project download process is more reliable with the features from this new version. In addition, this ADE release enhances some of the widgets with feedback from our developer community. The feedback gives us wonderful ideas on how to make the widgets more user-friendly and more aesthetical.

Please go to Download - ADE for downloading this new ADE release.
