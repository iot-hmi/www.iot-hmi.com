Immense Oak Technologies releases Acorn Development Environment(ADE) 3.2.0

Apr 26, 2021

Hsinchu, Taiwan – Immense Oak Technologies, Inc. (IOT) releases version 3.2.0 of Acorn Development Environment (ADE) This is a major release of ADE, which include a new feature and several bug fixes. The detail of this release is described below:

What’s new

  • Audio widget for loading and playing audio files under .wav format
  • Visibility control for the multi-layered images
  • Location and parameter/attribute lock for widgets

Bug Fixes

  • Able to control screen lock and the lock time
  • Register type check for wrong type selection
  • Adding positive sign in Digits Widget
  • Adding default file path for downloading firmware and loader
  • Making Image handler easier to use for position settings
  • Ctrl+F5 for refreshing document templates and recent designs
  • Pining and deleting recent designs
  • Button widgets’ body displayed correctly according to the body radius setting
  • Progress bar data value setting for all segments
  • Adhering to the aspect ratio for the images loaded from the Clipboard with the lock size attribute turned on
  • Enabling undo and redo for ALIGNMENT and SAME SIZE & SPACE control buttons
  • Fixing shared property display if two or more widgets are selected
  • Enhancing the built-in simulator response time

This new release of ADE is available immediately. Customers can download it by visiting our website . Customers who need technical support please contact us, support@iot-hmi.com, or visit our website.
